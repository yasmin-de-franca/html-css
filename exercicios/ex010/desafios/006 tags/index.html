Tags em HTML

Em HTML 5, podemos inserir os elementos da lista a seguir. Passe o mause sobre eles para descobrir qual tag deverá ser utilizada em cada caso.

  • Parágrafos
  • Títulos
  • Imagens
  • Lista numeradas
  • Listas com demarcadores
  • Marca Texto
  • CSS integrado a folha de HTML (geralmente para coisas pequenas)
